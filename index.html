El HEADER Se utiliza típicamente para representar la cabecera de una página web
La NAVEGACION Generalmente se utiliza para contener enlaces de navegación dentro de un sitio web

La etiqueta p Se utiliza para definir un párrafo de texto en una página web. Es una de las etiquetas más comunes y se utiliza para estructurar el contenido textual de manera semántica

El SECTION Se utiliza para dividir el contenido de una página web en secciones temáticas o agrupaciones lógicas de contenido. Cada sección debería representar un contenido autónomo o independiente
El ASIDE Se utiliza para representar contenido secundario o complementario dentro de una página web. Este contenido es generalmente relacionado con el contenido principal de la página, pero no es esencial para comprenderlo
El FOOTER Se utiliza para representar la sección de pie de página de una página web. Generalmente, esta sección contiene información de contacto, enlaces de navegación adicionales, créditos, derechos de autor, y otros elementos que son comunes en todas las páginas del sitio.
